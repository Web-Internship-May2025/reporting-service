# Reporting Service

The Reporting Service is responsible for generating all types of documents and storing them in the database.

## Functions of the Reporting Service
- **Generating reports, offers, insurance policies based on proposals** (Policy Creation System Service).
- **Generation of complaints, claim properties, decisions, etc.** (Claims Management System Service).
- **Storage of all documents submitted by subscribers to the Claims Management System Service**.
- **Sending documents and reports by email**.
- **Document access endpoint via document name**.

**Note:** Wherever document generation is mentioned, this service should be used.
